Colio - jQuery Portfolio Content Expander Plugin

About

Using Colio plugin you can display details about your portfolio items on the same page in form of expandable viewport that can be placed before, inside or after portfolio grid. Colio plugin comes in two versions - black and white, is compatible with isotope and quicksand filter plugins and works on tablets and mobile phones.

So give it a try, check examples below, and start adding Colio to your projects!
